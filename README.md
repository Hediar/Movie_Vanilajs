## 주요 기능 

### 영화 리스트 불러오기
- 영화 리스트를 보여줍니다.
- 제목, 별점, 설명을 볼 수 있습니다.

### 새로고침 했을 때
- 검색 하는 창에 자동으로 커서가 집중됩니다.

### 검색하기
- 검색하고 싶은 것을 검색할 수 있습니다.
- 일부 단어만 검색해도 검색이 가능합니다.
- 검색 버튼이나 엔터를 입력해도 작동이 됩니다.

### 영화 카드 클릭
- 영화 카드를 클릭할 수 있습니다.
- 클릭할 시 영화의 id를 보여줍니다.
